Notes

The network was trained on the MNIST corpus of 60,000 handwritten digit images introduced by LeCun et al. (1998). At inference, user-drawn strokes are cropped to their tight bounding box, scaled so the longest side spans 20 pixels, and re-centred within a 28 × 28 frame — matching the preprocessing convention of the original training set so the input distribution at test time is consistent with the distribution seen during training.

The argmax class is highlighted in vermilion; the remaining nine rows show the full posterior in monochrome. Reported latency measures the forward pass only, excluding the network round-trip and image decoding — values typically fall between 1 ms and 5 ms on Apple silicon (mps).

[1] LeCun, Y., Bottou, L., Bengio, Y., & Haffner, P. (1998). Gradient-based learning applied to document recognition. Proceedings of the IEEE, 86(11), 2278–2324.

[2] Loshchilov, I., & Hutter, F. (2019). Decoupled weight decay regularization. International Conference on Learning Representations.
